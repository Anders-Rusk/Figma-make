# Spacing Scale Documentation

## Overview
This document provides a comprehensive guide to the spacing scale used in our design system. The spacing scale is based on a modular scale and is designed to create consistent spacing throughout our UI designs. Below are the spacing tokens from Space/0 to Space/8x, including their values, usage guidelines, and preview information.

## Spacing Tokens

| Token   | Value   | Preview           | Usage Guidelines               |
|---------|---------|-------------------|--------------------------------|
| Space/0 | 0px     |                   | No spacing                     |
| Space/1 | 4px     | ░░░░              | Use for small elements         |
| Space/2 | 8px     | ░░░░░░░░░░       | Use for padding of buttons       |
| Space/3 | 16px    | ░░░░░░░░░░░░░░   | Base spacing for sections      |
| Space/4 | 24px    | ░░░░░░░░░░░░░░░░░░   | Use for margin between sections |
| Space/5 | 32px    | ░░░░░░░░░░░░░░░░░░░░   | Use for larger gaps            |
| Space/6 | 40px    | ░░░░░░░░░░░░░░░░░░░░░░   | Use between major components    |
| Space/7 | 48px    | ░░░░░░░░░░░░░░░░░░░░░░░░   | Use for large sections         |
| Space/8x| 64px    | ░░░░░░░░░░░░░░░░░░░░░░░░░░   | Use for extra large spacing    |

## Usage Examples
- **Space/1** can be used for small icons and buttons, ensuring they have sufficient padding.
- **Space/4** is ideal for spacing between form fields to enhance legibility.
- For larger elements, such as cards or sections, **Space/6** allows for clear separation and enhanced focus.

## Conclusion
Consistent use of these spacing tokens helps create harmony and balance in our designs. Always refer to this scale when designing new components to ensure adherence to our spacing guidelines.
