# Система Отступов

Система отступов использует стандартный набор токенов для обеспечения единообразного применения отступов в проекте.

## Токены
| Название токена | Значение | Описание |
|------------------|----------|----------|
| `gap-xs` | `4px`    | Минимальный отступ, используемый для небольших элементов. |
| `gap-sm` | `8px`    | Небольшой отступ для отделения элементов. |
| `gap-md` | `16px`   | Средний отступ для создания пространства между компонентами. |
| `gap-lg` | `24px`   | Большой отступ для значительного разделения. |
| `gap-xl` | `32px`   | Экстра большой отступ для больших зон пространства. |

## Подсказки по Использованию

### Отступы для Интервалов (Gap)
- Используйте `gap-xs` для наиболее компактных пространств.
- Применяйте `gap-md` для проверки равномерности отделения между компонентами.
- Используйте `gap-lg` и `gap-xl`, чтобы выделить ключевые элементы визуально.

### Отступы для Заполнения (Padding)
- Для элементов, которые требуют пространства внутри, например кнопки или карточки, используйте `padding` токены согласно эмоциональному эффекту, который вы хотите достичь.

### Отступы для Полей (Margin)
- Для создания четкого разделения между компонентами применяйте `margin` токены.

## Примеры
### Gap
```html
<div style="display: flex; gap: var(--gap-md);">
  <div>Элемент 1</div>
  <div>Элемент 2</div>
</div>
```
### Padding
```html
<div style="padding: var(--gap-sm);
  ">Содержимое с отступом</div>
```
### Margin
```html
<div style="margin-top: var(--gap-lg);
  ">Элемент с верхним отступом</div>
```

## Лучшие практики
- Применяйте отступы последовательно, чтобы сохранить единообразие в проекте.
- Используйте более крупные отступы для ключевых элементов интерфейса, чтобы привлечь внимание к важным разделам.
- Будьте внимательны к тому, как отступы влияют на общий баланс и компоновку ваших дизайнов.
