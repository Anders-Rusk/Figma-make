# Базовая шкала отступов

Базовые отступы, которые разрешено использовать в Gap (отступы между элементами/блоками) и в Padding (внутри элемента).

## Токены отступов

| Token          | Value  | Использование                       |
|----------------|--------|------------------------------------|
| Space/0       | 0 px   | Без отступа                         |
| Space/0,5x    | 2 px   | Минимальный отступ                  |
| Space/1x      | 4 px   | Очень маленький отступ             |
| Space/1,5x    | 6 px   | Маленький отступ                   |
| Space/2x      | 8 px   | Маленький отступ                   |
| Space/2,5x    | 10 px  | Маленький-средний отступ           |
| Space/3x      | 12 px  | Маленький-средний отступ           |
| Space/3,5x    | 14 px  | Средний отступ                     |
| Space/4x      | 16 px  | Средний отступ                     |
| Space/5x      | 20 px  | Большой отступ                     |
| Space/6x      | 24 px  | Большой отступ                     |
| Space/7x      | 28 px  | Очень большой отступ               |
| Space/8x      | 32 px  | Очень большой отступ               |

## Лучшие практики

- Применяйте отступы последовательно для единообразия
- Используйте более крупные отступы для выделения важных разделов
- Следуйте масштабу 4px для поддержания гармонии дизайна
